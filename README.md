# Future Interns - Task 2: Client Lead Management System (Mini CRM)

A full-stack Mini CRM built as part of the Future Interns Full Stack Web Development Program.

Features
- Add new client leads
- View lead details in modal
- Update lead status (New / Contacted / Converted)
- Add notes & follow-ups
- Logout (Admin) button
- Clean and responsive UI

 Tech Stack
- **Backend**: Node.js + Express
- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Storage**: In-memory (ready for MongoDB)

How to Run
1. `cd backend`
2. `npm install`
3. `node server.js`
4. Open `frontend/index.html` in your browser
